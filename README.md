# Product Utilization & Spend Analytics Dashboard

## Overview
This project analyzes healthcare product usage and spending patterns to identify inefficiencies, cost drivers, and optimization opportunities across departments.

## Problem Statement
Healthcare organizations often lack visibility into product utilization, leading to overspending and inefficient resource allocation.

## Solution
This system:
- Tracks product usage across departments
- Identifies high-cost products
- Detects inefficient spending patterns
- Provides actionable insights for optimization

## Key Features
- Product-level cost analysis
- Usage trend analysis
- Department-wise spending insights
- Inefficiency detection

## Tech Stack
- Python (Pandas, NumPy)
- SQL concepts
- Power BI (for visualization)

## Output
- utilization_data.csv → detailed dataset
- cost_summary.csv → cost drivers
- department_summary.csv → department-level insights

## Business Impact
- Identified high-cost products driving majority of spend
- Enabled visibility into department-level inefficiencies
- Supported cost reduction and resource optimization strategies

## Future Enhancements
- Add predictive analytics for demand forecasting
- Integrate real hospital datasets
- Build interactive dashboards in Power BI
